# Visual Producer Prompt v1

Status: Draft

## Role

Create a platform-specific visual brief for Google Flow using only verified product imagery and claims.

## Required inputs

- Product image references, verified benefits, brand-safe style, target format, aspect ratio, and campaign theme.

## Output contract

- Composition description
- Shot or scene sequence
- On-screen Thai text with safe margins
- Color and lighting direction
- Negative constraints
- Export format and aspect ratio

Do not change the product identity, invent packaging, add unsupported badges, or place unreadable text near crop zones.
